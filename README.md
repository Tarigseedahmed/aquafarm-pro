# Aquafarm Pro

نظام متكامل لإدارة المزارع المائية

## الميزات

- مراقبة جودة المياه
- إدارة الأسماك والإنتاج
- تتبع الغذاء والتغذية
- تقارير ولوحات تحكم
- إدارة الموظفين والمهام

## التقنيات المستخدمة

- Frontend: React + TypeScript
- Backend: Node.js + Express
- Database: PostgreSQL
- Mobile: React Native

## التثبيت

```bash
# تثبيت المكتبات
npm install

# تشغيل التطبيق
npm start
```
