## 🎬 سيناريوهات الاستخدام السريعة

### 🔐 تسجيل مستخدم جديد

1.افتح: http://localhost:3001
2. انقر "Sign Up" أو "تسجيل"
3. أدخل البيانات
4. تحقق من البريد في: http://localhost:8025

### 🧪 تجربة API

1.افتح: http://localhost:3000/api/docs
2. اختر أي Endpoint
3. انقر "Try it out"
4. أدخل البيانات
5. انقر "Execute"

### 📊 عرض البيانات في قاعدة البيانات

1.افتح: http://localhost:5050
2. سجل دخول بالبيانات المذكورة
3. أضف Server جديد:
   -Host: `postgres-dev`
   -Port: `5432`
   -Database: `aquapro_dev`
   -Username: `postgres`
   -Password: `admin123`

## 🔧 أوامر سريعة

### عرض Logs

```powershell
docker-compose -f docker-compose.dev.yml logs -f
```

### عرض حالة الخدمات

```powershell
docker-compose -f docker-compose.dev.yml ps
```

### إعادة تشغيل الخدمات

```powershell
docker-compose -f docker-compose.dev.yml restart
```
